# Metodo sito geturn{#geturn-site-method}

Questo metodo restituisce l'URN per questo sito.

## Esempio Java {#section_nyl_ycs_rz}

Utilizzo:

```
site.getUrn();
```

Output di esempio:

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```
