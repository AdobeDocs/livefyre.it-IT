# getUrn Site Method{#geturn-site-method}

Questo metodo restituisce l’URL del sito.

## Esempio Java {#section_nyl_ycs_rz}

Utilizzo:

```
site.getUrn();
```

Output campione:

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```
