# Ricerca per utenti{#search-for-users}

>
>Questi campi supportano la ricerca con caratteri jolly.

1. Solo elenco **[!UICONTROL Banned]** o **[!UICONTROL Whitelisted]** utenti.
1. Elenca solo **[!UICONTROL Studio]** gli utenti.
