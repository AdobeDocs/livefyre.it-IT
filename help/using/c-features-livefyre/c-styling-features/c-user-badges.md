# Badge utente{#user-badges}

Utilizza i CSS con tag utente per creare i badge utente per i membri della tua community.

I badge utente sono funzioni di stile personalizzate che ti consentono di personalizzare lo stile dei contenuti Livefyre pubblicati da tipi di utenti specifici (come moderatori, editor o qualsiasi altro gruppo di utenti che desideri definire) in Chat Livefyre, Commenti, Recensioni o App di blog live. Questo consente di evidenziare questi contenuti agli utenti.

App che utilizzano questa funzione:

* [Chat](../../c-about-apps/c-chat-app/c-chat-app.md#c_chat_app)
* [Commenti](/help/using/c-about-apps/c-comments/c-comments.md)
* [Recensioni](../../c-about-apps/c-reviews-app/c-reviews-app.md#c_reviews_app)
* [Note](../../c-about-apps/c-sidenotes-app/c-sidenotes-app.md#c_sidenotes_app)
